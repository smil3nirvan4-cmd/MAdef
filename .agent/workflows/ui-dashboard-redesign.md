Objetivo: reorganizar o dashboard para clareza, consistência e estética moderna.

Tarefas:
1) Reorganizar o header (banner):
   - melhorar contraste do texto
   - opcional: gradiente sutil (apenas se mantiver AA)
2) KPI cards:
   - tipografia e espaçamento padronizados
   - estados vazios: quando 0, mostrar dica/CTA
3) “Módulos do Sistema”:
   - cards uniformes (ícone, título, descrição, affordance de clique)
   - hover consistente
4) Garantir alinhamentos e ritmo visual.
5) Validar com Browser + gerar:
   - Screenshot AFTER
   - Recording AFTER (scroll + clique em 1 módulo)

Critérios de aceite:
- Layout mais coerente e legível.
- Zero textos “apagados”/sem contraste.
- Antes/depois documentado.
