Objetivo: reestruturar o layout global do Admin (sidebar + topbar + content) com UX melhor.

Tarefas:
1) Melhorar hierarchy/spacing:
   - grid consistente
   - padding de página e seções
2) Sidebar:
   - estados claros (hover/active/focus)
   - agrupamento por seções
   - reduzir ruído visual
3) Topbar:
   - search mais claro e acessível
   - status/actions alinhados
4) Responsividade:
   - em largura menor: sidebar colapsável (se aplicável)
5) Entregar walkthrough com:
   - mudanças feitas
   - como testar.

Critérios de aceite:
- Navegação parece consistente e “premium”.
- Focus e contraste OK.
- Browser recording mostrando navegação pós-mudança.
