Objetivo: definir um sistema tipográfico B2B admin (legível, consistente, pt-BR).

Tarefas:
1) Confirmar fonte atual (provável Inter). Manter se já estiver instalada.
2) Definir escala tipográfica:
   - H1/H2/H3
   - CardTitle
   - Body (default)
   - Label
   - Caption/Helper
3) Definir pesos e line-height por nível.
4) Definir regras:
   - quando usar muted-foreground
   - como tratar links
   - densidade de informação em cards e sidebar.
5) Entregar:
   - docs/ui-audit/02-typography.md (guia)
   - tokens CSS (variables) e mapeamento Tailwind.

Critérios de aceite:
- Hierarquia visual clara no dashboard.
- Texto secundário continua legível (sem cinza “apagado”).
